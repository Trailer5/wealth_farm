# Agent Docs

本目录存放 Agent 按场景读取的项目规则和工作规范。

## Start Here

- `.agent/AGENT_README.md` 是每轮固定必读 memo。
- `.agent/SKILL.md` 是 Agent 路由索引。
- 本文件只在 `.agent/docs/` 文件职责不清时阅读。

## Files

- `development_rule.md`：项目开发规则，负责目录职责、文件位置、协作边界和版本兼容。
- `development_report_rule.md`：开发计划和开发汇报规则，负责文末 tracking checklist 的格式和维护方式。

## Rule Of Thumb

先读最小相关文档。不要为了保险一次性读取所有 `.agent/docs/` 文件。
